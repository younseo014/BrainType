# [PRD v2-B] 브레인타입 (BrainType) — 주변인 평가 기반 뇌 강점 탐색 플랫폼

> **방향성 요약**: "나의 문제를 찾는 앱"이 아닌 "나의 뇌 강점을 발견하는 앱"으로 프레이밍한다. 본인이 아닌 주변인(가족·동료·단골 손님)이 평가자가 되어 더 객관적인 데이터를 수집하고, MBTI처럼 공유 가능한 "두뇌 유형" 결과를 제공해 자발적 확산을 유도한다.

---

## 반영된 피드백 항목

| 피드백 출처 | 내용 | 반영 방식 |
|---|---|---|
| 교수님 공통 #1 | 음성보다 시각적 UI 선호 | 전 기능 시각 중심 UI. 음성 기능 없음. |
| 교수님 공통 #2 | 진단으로 연결될 수 있는가 | 강점 탐색 결과가 자연스럽게 약점을 노출 → 전문 상담 연계 |
| 교수님 공통 #4 | 기만하지 않되 치매 앱임을 숨기기 | "두뇌 유형 테스트" 프레이밍으로 시작, 후반부에 인지 건강 연계 고지 |
| 이승은 교수님 | 강점을 찾다 보면 약점도 드러남 | 강점 중심 결과지 제공, 약점은 부드럽게 표시 |
| 이승은 교수님 | 본인보다 주변인 평가가 더 정확 | 피평가자 초대 → 주변인이 관찰 기반 설문 작성 |
| 팀 아이디에이션 | 포인트 리워드 | 설문 완료 시 양측(평가자·피평가자) 포인트 지급 |
| 팀 아이디에이션 | 2단계 진단 구조 | 1단계(주변인 관찰 설문) → 2단계(피평가자 직접 인지 테스트) |

## 1. 제품 개요

- **제품명**: 브레인타입 (BrainType)
- **한 줄 설명**: 주변 사람들이 나의 두뇌 강점을 발견해 주는 관계 기반 인지 건강 플랫폼.
- **핵심 가치 제안**: "MBTI처럼 공유하고 싶은 나의 두뇌 유형" — 낙인 없이 자발적으로 참여하고, 주변인의 관찰로 더 정확한 인지 현황을 파악한다.
- **타깃 플랫폼**: 모바일 웹 (카카오/네이버 계정 소셜 로그인) → 초기 배포 장벽 최소화

---

## 2. 서비스 구조 — 두 개의 역할

브레인타입은 **피평가자(당사자)**와 **평가자(주변인)** 두 역할로 구분된다.

```
피평가자 (당사자)
  → 앱 설치 및 "두뇌 유형 탐색" 시작
  → 주변인에게 초대 링크/코드 발송
  → 평가자들의 관찰 결과 취합 후 두뇌 유형 리포트 수신
  → 2단계 테스트 권유 (선택)

평가자 (가족·동료·단골 손님 등)
  → 링크 클릭 → 간단한 관찰 설문 작성 (5~7분)
  → 완료 시 포인트 지급 (본인 계정 생성 선택)
  → 피평가자 결과 공개 시 함께 결과 확인 가능
```

## 3. 타깃 사용자 & 페르소나

### 페르소나 A — 피평가자: 56세 자영업자 (남)
- **상황**: 본인은 "요즘 좀 피곤해서 그런 거"라고 생각하지만, 딸이 아버지의 반복 질문과 최근 건망증이 걱정됨.
- **진입 경로**: 딸이 "아빠, 이 테스트 재밌어. 두뇌 유형이 뭔지 나와!" 하며 초대 링크를 전송.
- **욕구**: 치매 검사라는 느낌 없이 가볍게 참여. 결과가 긍정적으로 표현되길 원함.

### 페르소나 B — 평가자: 28세 직장인 (딸)
- **상황**: 부모님의 인지 변화를 감지했지만 직접 "치매 검사 받으세요"라고 말하기 부담스러움.
- **욕구**: 낮은 진입 장벽으로 부모님의 인지 상태를 객관적으로 확인하고 싶음.

### 페르소나 C — 평가자: 단골 손님 / 직장 동료
- **상황**: 알게 모르게 당사자의 변화(반복 질문, 거스름돈 실수)를 관찰해 온 사람.
- **욕구**: 짧은 설문으로 "좋은 일"에 참여한다는 느낌. 소소한 보상이 있으면 참여 의지 상승.

## 4. 핵심 기능 상세

### ① 두뇌 강점 유형 프레임 (핵심 프레이밍)

기존 "인지 저하 탐지"를 **8가지 두뇌 강점 유형**으로 재구성:

| 유형명 | 강점 설명 | 내포된 인지 영역 |
|---|---|---|
| 기억 장인형 | 오래된 기억을 생생하게 저장 | 장기 기억 |
| 순발력형 | 새로운 정보를 빠르게 처리 | 처리 속도 |
| 언어 감각형 | 단어와 표현이 풍부 | 언어 유창성 |
| 공간 탐험형 | 지도 없이도 길을 잘 찾음 | 공간 지각 |
| 집중 몰입형 | 한 가지에 깊이 집중 | 지속 주의력 |
| 멀티태스킹형 | 여러 일을 동시에 처리 | 작업 기억 |
| 감정 공감형 | 상대의 감정을 빠르게 읽음 | 사회적 인지 |
| 신중 계획형 | 실수 없이 꼼꼼하게 계획 | 실행 기능 |

- 결과지는 강점 상위 2개 유형을 전면에 부각.
- 상대적으로 낮은 영역은 "더 키워볼 수 있는 강점"으로 소프트하게 표시.

### ② 주변인 관찰 설문 (평가자 기능)
- **설문 형식**: 시각적 선택지 (텍스트 최소화, 이모지·일러스트 활용).
- **설문 항목 예시** (총 20문항, 5~7분 소요):
  - "최근 같은 이야기를 반복해서 하신 적이 있나요?" (빈도: 없음 / 가끔 / 자주)
  - "물건을 놓고 어디 뒀는지 못 찾는 경우가 있나요?"
  - "익숙한 사람 이름이나 단어를 바로 떠올리지 못하는 경우가 있나요?"
  - "새로운 기기나 방법을 배우는 데 전보다 어려움이 있어 보이나요?"
- **응답자 수**: 최소 2명 이상 응답 시 리포트 생성. 최대 10명까지 초대 가능.
- **평균화 처리**: 여러 평가자의 응답을 가중 평균하여 편향 최소화.

### ③ 포인트 리워드
- 평가자가 설문 완료 시: **100포인트** (계정 신규 생성 시 추가 50포인트)
- 피평가자가 리포트 확인 시: **50포인트**
- 2단계 테스트 완료 시: **200포인트**
- 결과 카드 SNS 공유 시: **30포인트** (바이럴 확산 유도)
- 전환: 네이버페이 / 카카오페이 (1,000포인트 = 1,000원)

### ④ 두뇌 유형 결과 리포트 (피평가자용)
- **시각 디자인**: MBTI 스타일 공유 카드 형태. SNS 공유 최적화.
- **구성**:
  - 나의 두뇌 유형 (상위 2개 강점 유형 + 유형 설명)
  - 강점 레이더 차트 (8개 영역)
  - 평가자 수 및 신뢰도 표시
  - "이런 직업/활동이 잘 맞아요" 추천
- **약점 노출 방식**: "○○ 강점을 더 키우면 ○○ 능력이 함께 올라가요" 형태로 약점을 간접 언급.

### ⑤ 2단계 직접 인지 테스트 (선택)
- **진입 트리거**: 주변인 평가에서 특정 영역 점수가 기준치 이하일 경우 소프트 권유.
- **테스트 구성** (약 5분):
  - 단어-그림 연결 기억 테스트
  - 숫자 역순 입력
  - 색깔-단어 스트룹 테스트 (시각적)
- **프레이밍**: "두뇌 유형 정밀 측정 모드" — "치매 검사"라는 표현 사용 안 함.
- **결과 활용**: 2단계 완료 시 리포트에 "정밀 측정 결과" 배지 추가 + 전문 상담 연계 옵션 제공.

### ⑥ 전문 상담 연계
- 2단계 결과 기준 점수 미달 시: "가까운 치매안심센터 무료 프로그램 안내" 팝업.
- 사용자가 원하면 현재 위치 기반으로 가장 가까운 치매안심센터 지도 연결.
- "주변 사람들이 걱정하고 있어요" 대신 "두뇌 강점을 더 키울 수 있는 전문 프로그램이 있어요"라는 메시지.

## 5. 확산 전략 — 바이럴 루프

```
피평가자 (부모/지인)
       ↓ 초대 링크 발송
평가자 (자녀/동료/손님)
       ↓ 설문 완료 → 포인트 지급
       ↓ 결과 카드 SNS 공유 → 신규 사용자 유입
       ↓ 신규 사용자가 다시 피평가자 또는 평가자로 진입
```

- 핵심 바이럴 포인트: **공유 가능한 두뇌 유형 카드**. "내 두뇌 유형은 기억 장인형이래!" 형태로 SNS 자발 공유 유도.

## 6. 온보딩 — 점진적 고지 구조

```
[1단계] 랜딩 페이지
  → "당신의 두뇌 강점 유형을 알아보세요. 주변 사람들이 당신을 어떻게 보는지 확인해 보세요."
  → 치매/인지저하 언급 없음.

[2단계] 평가자 초대 및 설문 완료 후
  → "브레인타입은 두뇌 강점 탐색과 함께 인지 건강 모니터링을 지원해요.
     지속적인 건강 관리를 원하시면 알림을 켜 두세요."
  → 선택적 동의.

[3단계] 2단계 테스트 권유 시
  → "더 정확한 두뇌 유형 측정을 위한 5분 테스트를 권장해요.
     이 테스트는 인지 기능 전반을 점검하는 방식으로 구성돼 있어요."
  → 이 시점에 인지 검사임을 명시적으로 고지.
```

## 7. MVP 범위 및 성공 지표

### MVP (4주 개발 목표)
- 피평가자 초대 코드 생성 및 공유
- 평가자 관찰 설문 (20문항) 및 결과 취합
- 두뇌 유형 결과 카드 생성 및 SNS 공유
- 포인트 적립 로직 (전환은 v2)

### 성공 지표

| 지표 | 목표치 | 측정 방법 |
|---|---|---|
| 피평가자당 평균 평가자 수 | 3명 이상 | 초대 완료율 |
| 결과 카드 SNS 공유율 | 30% 이상 | 공유 버튼 클릭 |
| 2단계 테스트 참여율 | 25% 이상 | 이벤트 로그 |
| 치매안심센터 연계 클릭률 | 10% 이상 | 딥링크 클릭 수 |
| 바이럴 계수 (K-factor) | 1.2 이상 | 초대를 통한 신규 가입 비율 |
